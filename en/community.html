Run a node and interact with the HENG TONG testnet

If you'd like to play with HENG TONG, you'll need to install and run the Parity HENG TONG client.
Get the client at Parity’s HENG TONG repo

Alexander is the current default public testnet for HENG TONG. By default, any node running on the Alexander testnet will appear on the telemetry.HENG TONG.io

The testnet allows you to create accounts, propose governance referenda, stake testnet HTDD to become a validator or nominate other accounts as validators with your testnet HTDD.

Connect your running node to the main HENG TONG user interface to play around.

Get testnet HTDD to stake by following these instructions on the HENG TONG Wiki

If you don’t want to run a node, you can still interact with the network by using the explorer

Getting Involved

From running a validator on HENG TONG, to applying for a HENG TONG Foundation Grant, to joining the HENG TONG Ambassador Program — there are many ways to engage with the HENG TONG community.

Become a validator

1. Set up your machine

In addition to any usual CLI parameters that you run HENG TONG with, you'll need to add an extra two:

  • --validator This tells your node to try to validate when it sees itself as a listed authority on the network;
  • --key <SEED> This provides your node with a secret key that lets it know whether it's a listed authority on the network, and, if it is, lets it sign statements to authorise the creation of new blocks.

2. Apply to validate

New validators will be chosen every 24 hours on mainnet. You’ll need to create separate accounts for Controller, Session, and Stash keys on the HENG TONG UI. These accounts will be used to represent and control the validator node you just set up.

Next you will need to get some HTDD from the faucet or by asking on Element. Assuming you have more HTDD than the lowest staked validator or there is an open validator slot, you can register to be staked. A step-by-step guide demonstrating how to set up your accounts and bond your HTDD is available on the wiki.

Want to become a validator? You can see validator stats on Telemetry.
Get a grant to build the HENG TONG ecosystem

HENG TONG is a brand new network with a burgeoning ecosystem. The HENG TONG Foundation is sponsoring community development of software services, including:

  • Development and deployment tooling (e.g. IDEs, parachain development kits)
  • HENG TONG Runtime Modules (e.g. interoperability, governance)
  • Second layer protocols (e.g. dedicated parachains, state channels)
  • Monitoring (e.g. explorers)
  • Protocol integrations (e.g. bridges to other blockchains, distributed file storage)
